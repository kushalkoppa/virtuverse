# VirtuSpace

VirtuSpace is the unified platform that integrates EnviHub and PlantHub, providing seamless access to both virtualization and plant simulation tools.

## Overview

VirtuSpace serves as the integration layer for:
- **EnviHub**: Virtualization and simulation tools
- **PlantHub**: Plant simulation and manufacturing tools

## Features

- Unified navigation between EnviHub and PlantHub
- Centralized dashboard showing metrics from both platforms
- Seamless tool switching
- Integrated model management

## Architecture

VirtuSpace acts as a container application that:
- Hosts both EnviHub and PlantHub frontends
- Provides unified navigation
- Aggregates data from both platforms
